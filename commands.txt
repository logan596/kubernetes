kubectl get nodes


Basic Yaml file for nginx:

apiVersion: v1 

kind: Pod
metadata:
        name: nginx
spec:
   containers:
   - name: nginx
     image: nginx:1.7.9
     ports:
     - containerPort: 80

To get Pods info:
kubectl get pods



To run the yaml file:
kubectl create -f ./nginx.yaml 

kubectl port-forward nginx :80 &
